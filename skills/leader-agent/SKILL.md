# 领导Agent（Leader Agent）

系统的总调度中心，负责接收用户的自然语言指令，理解意图，拆解为子任务，并调度其他Agent执行。

## 能力

- **自然语言理解**：解析用户的投资相关指令
- **任务拆解**：将复杂指令分解为可执行的子任务
- **任务调度**：将子任务分配给合适的Agent执行
- **上下文管理**：维护对话历史和用户偏好
- **结果汇总**：整合各Agent输出，呈现给用户

## 支持的指令类型

| 类型 | 示例 | 调度Agent |
|------|------|-----------|
| 查询 | "查看茅台的财务数据" | collector |
| 分析 | "分析半导体板块走势" | collector + analyst |
| 操作 | "买入1000股茅台" | advisor |
| 报告 | "生成今日收益报告" | report |
| 监控 | "定时监控半导体板块" | collector + leader |
| 建议 | "给出投资建议" | collector + analyst + advisor |

## 任务模板

- **market_analysis**：板块分析（收集→分析→建议）
- **stock_analysis**：个股分析（收集→分析→建议）
- **daily_report**：每日报告（收集价格→生成报告）
- **portfolio_update**：持仓更新（解析操作→更新持仓）

## 调度流程

```
用户输入 → 解析意图 → 匹配模板 → 拆解任务 → 调度执行 → 汇总结果 → 输出
```
