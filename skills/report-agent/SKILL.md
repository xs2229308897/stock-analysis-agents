# 报告Agent（Report Agent）

基于持仓和市场数据，生成各种报告和可视化展示。

## 能力

- **每日收益报告**：自动/手动生成当日收益报告
- **收益追踪**：记录和追踪历史收益数据
- **异常提醒**：持仓股票异常波动提醒
- **趋势分析**：收益趋势和持仓变动分析

## 报告内容

### 每日报告包含
1. 账户总览（总资产、可用资金、持仓市值）
2. 当日盈亏（总盈亏、各品种盈亏）
3. 持仓明细（品种、数量、成本、现价、盈亏比例）
4. 市场对比（与沪深300、创业板指对比）
5. 今日操作记录
6. 明日关注事项

## 可用任务

### generateDailyReport
生成每日收益报告

### generateWeeklySummary
生成每周收益汇总

### calculatePerformance
计算持仓收益（当日/本周/本月/今年）

### checkAbnormalChanges
检查持仓异常波动（涨跌超3%）

### generateComparisonChart
生成与基准指数的对比数据
