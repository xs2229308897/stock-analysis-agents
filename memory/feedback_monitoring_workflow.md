## 盘中监控流程

### 三步并行获取（已验证有效）
```
并行调用：
├── node tools/realtime-quote.js          → 大盘指数
├── node tools/realtime-quote.js --board   → 板块 ETF（全部18个）
└── node tools/realtime-quote.js shXXX...  → 重点个股
```

### 全板块展示
- 用户明确要求"获取所有板块的信息"，不再只展示涨跌幅前几名
- 趋势分析中只重点标注当日核心板块的变化

### 趋势对比规范
- 每次刷新需对比上次数据，标注变化趋势
- 趋势标注：加速 ↑ / 减速 ↓ / 翻转 ↕ / 持平 →
- 异常信号：单板块涨跌超3%、大盘变动超1%、个股超5%

### 异常信号体系（2026-05-26验证）
- 今日共触发8项异常信号（历史最高），14:02达到峰值
- 异常信号可全部解除（今日14:42全部解除）
- 科创50/半导体ETF/计算机ETF/军工ETF为主要异常触发板块

### 午间休市处理
- 11:30-13:00为午间休市，数据不会更新
- 用户多次在休市期间触发监控，应主动提示"午休中，13:00后再触发"

### 定时任务
- 使用 CronCreate 设置定时监控（推荐10分钟间隔）
- 收盘后（15:00）应提醒用户取消定时任务
- 单次定时任务（recurring: false）适合午间暂停后恢复

**Why:** 2026-05-26全天监控过程中，用户多次给出反馈优化监控流程
**How to apply:** 后续盘中监控严格执行此流程，特别是全板块展示和午间休市提示
